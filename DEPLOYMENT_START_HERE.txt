### PHASE 2: Render (5 minutes)

1. Go to https://render.com
2. Sign up with GitHub
3. Click "+ New" ‚Üí "Web Service"
4. Connect your repo
5. Fill settings:
   - Root: `FACE_DETECTION`
   - Build: `pip install -r requirements.txt`
   - Start: `python app.py`
6. Add environment variable:
   - KEY: `PORT`
   - VALUE: `10000`
7. Click "Create Web Service"
8. Wait 5 minutes for deployment

### PHASE 3: Verify & Save (2 minutes)

1. Render shows your live URL
2. Test it by clicking the URL
3. Upload a photo and verify it works
4. Save the URL to `link_web_app.txt`
5. Push to GitHub

```powershell

git commit -m "Add URL"
git push
```
