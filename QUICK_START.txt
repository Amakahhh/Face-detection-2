================================================================================
TROUBLESHOOTING
================================================================================

Problem: "No module named tensorflow"
Solution: Run: pip install -r requirements.txt

Problem: "Model file not found"
Solution: First run python model_training.py

Problem: "No face detected"
Solution: Use a clear, frontal face image with good lighting

Problem: "Port 5000 already in use"
Solution: Close other Flask apps or change port in app.py

Problem: Training takes too long
Solution: Normal! CNN training takes 5-10 minutes. Be patient.

================================================================================
DEPLOYMENT (Optional - to Render)
================================================================================

When ready to deploy:

1. Create GitHub repository with FACE_DETECTION folder
2. Go to render.com and create new Web Service
3. Connect GitHub repo
4. Set build command: pip install -r requirements.txt
5. Set start command: python app.py
6. Add PORT environment variable: 10000
7. Deploy!

Then copy your deployment URL to link_web_app.txt
